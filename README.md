# 🚀 Coleção de Exercícios: Fundamentos de Java e Estruturas de Dados

Este repositório contém uma série de exercícios práticos desenvolvidos em **Java**, focados no domínio de vetores (arrays), estruturas de repetição (`for`, `do-while`) e lógica de processamento de dados.

---

## 📂 Lista de Exercícios e Funcionalidades

A coleção aborda diferentes níveis de manipulação de dados, desde operações aritméticas simples até a filtragem complexa de caracteres:

* **Exercicio_01**: Leitura e exibição de um vetor de 5 números inteiros.
* **Exercicio_02**: Manipulação de ordem, realizando a leitura de 10 números e exibindo-os na ordem inversa.
* **Exercicio_03**: Sistema acadêmico simples que recebe 4 notas, calcula a média e exibe os valores originais.
* **Exercicio_04**: Analisador de caracteres que identifica e conta quantas consoantes existem em uma string de 10 posições, listando-as individualmente.
* **Exercicio_05**: Separador de paridade que lê 20 números e os organiza em dois vetores distintos (Pares e Ímpares).
* **Exercicio_06**: Gerenciador de médias escolares para 10 alunos. Calcula a média aritmética de 4 notas por aluno e lista apenas aqueles aprovados (média >= 7.0).
* **Exercicio_07**: Processamento matemático de um conjunto de 5 números, calculando simultaneamente a soma e a multiplicação de todos os elementos.

---

## 🛠️ Tecnologias e Conceitos Aplicados

* **Linguagem**: Java.
* **Entrada de Dados**: Uso intensivo da classe `Scanner`.
* **Estruturas de Dados**: Vetores unidimensionais (arrays) para armazenamento temporário.
* **Controle de Fluxo**: 
    * Laços `for` e `foreach` para percorrer coleções.
    * Condicionais `if/else` para filtragem de regras de negócio (aprovação, paridade, identificação de consoantes).
* **Tipagem**: Uso de `int`, `double` e `long` para garantir a precisão em cálculos de grande escala.

---

## 🚀 Como Executar

1.  Escolha o exercício desejado (ex: `Exercicio_07.java`).
2.  Compile o arquivo no terminal:
    ```bash
    javac Exercicio_07.java
    ```
3.  Execute a aplicação:
    ```bash
    java Exercicio_07
    ```
